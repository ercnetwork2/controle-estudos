Como publicar no GitHub Pages
  1. Crie um repositório novo no GitHub, por exemplo controle-de-estudos.
  2. No seu projeto (Vite/React, Next export estático ou simples HTML com CDN), inclua este componente como página principal.
  3. Faça commit e push.
  4. No repositório, vá em Settings → Pages e selecione a branch main e a pasta / (root) ou /docs.
  5. Abra a URL exibida pelo GitHub Pages (algo como https://seu-usuario.github.io/controle-de-estudos).
Importando sua planilha existente

Use o botão Importar e selecione seu arquivo .xlsx (ou .csv/.json). O app tentará reconhecer cabeçalhos como Data, Matéria, Tópico, Horas, Status e Notas.

Backup & sincronização via Git
